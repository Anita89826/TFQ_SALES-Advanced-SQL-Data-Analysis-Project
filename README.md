# TFQ_SALES-Advanced-SQL-Data-Analysis-Project
The project is built on three main tables:  - **employee** – Contains employee details such as `emp_ID`, `emp_NAME`, `DEPT_NAME`, and `salary`. - **dept** – Stores department information like `dept_ID` and `dept_name`. - **sales** – Includes transaction-level sales data with `store_id`, `store_name`, `cost`, and `quantit`
